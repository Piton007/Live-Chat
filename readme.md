# Chat en tiempo real 
### El presente proyecto usa tecnologia Websockets para la comunicacion en tiempo real. El Stack usado es el siguiente: Node TS, React Ts, Jest Ts y Socket IO 

## Pasos para iniciar
### 1. Iniciar Backend 
```
cd /Server
npm install
npm start
```
### 2. Iniciar Frontend
```
cd /Client
npm install
npm start
```

## Pasos para ejecutar tests
### 1. Iniciar Backend 
```
cd /Server
npm install
npm test
```
